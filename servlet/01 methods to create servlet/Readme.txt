This folder has implementation of all three methods to create a servlet.

These ways are
1.) By implementing the Servlet interface.
2.) By inheriting the GenericServlet class.
3.) By inheriting the HttpServlet class (widely used)

For compiling a servlet, jar file is required to be loaded.
A server can provide the said jar file.

Also it has brief description of lifecycle and non-lifecycle methods

The deployment descriptor is an xml file from which web container gets the information about the servlet to be invoked.
